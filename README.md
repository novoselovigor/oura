# 🍏 Панель Oura Ring & AI-Коуч: Инструкция по локальному запуску на macOS

Это руководство поможет вам скачать, настроить и запустить приложение на вашем Mac из Терминала. Приложение поддерживает два режима запуска:
1. **Full-Stack сервер** (с Node.js & Express) — поддерживает безопасное хранение API ключей на бэкенде и автоматический OAuth вход.
2. **Полностью статическая версия (Client-Only)** — идеальна для бесплатного хостинга на **GitHub Pages**, где все операции с Oura Cloud API и ИИ Идут напрямую из вашего браузера (localStorage).

---

## 📥 Шаг 1: Как забрать код на Mac

Вы можете перенести код на свой Mac двумя способами:
* **Через экспорт в Google AI Studio**: Нажмите на иконку настроек (шестеренка в верхнем углу панели AI Studio) ➔ Выберите **«Export as ZIP»** (Скачать архивом) или **«Export to GitHub»** (экспортировать в свой репозиторий).
* **Распакуйте ZIP** в любую удобную папку (например, на рабочий стол или в папку `Projects`).

---

## 🛠️ Шаг 2: Подготовка macOS терминала

Для запуска веб-приложений вам понадобится среда выполнения **Node.js** (версия 20 или новее).

1. Откройте встроенное приложение **«Терминал»** (Terminal) на macOS. Его можно найти через поиск Spotlight (`Cmd + Space` ➔ введите `Терминал`).
2. Проверьте, установлен ли у вас Node.js, выполнив команду:
   ```bash
   node -v
   ```
3. Если команда не найдена, установите Node.js одним из способов:
   * **Простой способ**: Скачайте официальный установщик *LTS* для macOS с сайта [nodejs.org](https://nodejs.org/).
   * **Для продвинутых (через Homebrew)**: Если у вас установлен пакетный менеджер Homebrew, выполните в терминале:
     ```bash
     brew install node
     ```

---

## 📦 Шаг 3: Переход в папку и установка зависимостей

1. В Терминале перейдите в папку с распакованным проектом. Например, если папка называется `oura-pwa` и лежит на Рабочем столе:
   ```bash
   cd ~/Desktop/oura-pwa
   ```
2. Установите все библиотеки и зависимости проекта:
   ```bash
   npm install
   ```

---

## ⚙️ Шаг 4: Настройка локальных ключей (.env)

Для полноценной работы с сервером создайте файл конфигурации для ваших API-ключей:

1. Сделайте копию шаблона настроек:
   ```bash
   cp .env.example .env
   ```
2. Откройте созданный файл `.env` в любом текстовом редакторе (например, встроенном TextEdit или VS Code):
   ```bash
   open -e .env
   ```
3. Заполните переменные:
   * `GEMINI_API_KEY`: Ваш персональный API ключ для ИИ-коуча. Получить его бесплатно можно за 1 минуту в личном кабинете [Google AI Studio](https://aistudio.google.com/).
   * Если вы планируете настроить автоматический OAuth вход Oura Ring (а не вводить персональные токены):
     * Создайте проект на [Oura Cloud API Developer Console](https://cloud.ouraring.com/settings/api).
     * Задайте Redirect URI как: `http://localhost:3000/auth/callback`
     * Скопируйте полученные `OURA_CLIENT_ID` и `OURA_CLIENT_SECRET` в ваш `.env`.

---

## 🚀 Шаг 5: Запуск!

### Вариант А: Локальный запуск для разработки (Fast Dev Mode)
Запускает сервер и клиент с поддержкой мгновенного обновления при изменении кода.
```bash
npm run dev
```
* После этого откройте браузер по адресу: **`http://localhost:3000`**
* Все готово! Вы можете тестировать, загружать архивы экспорта Oura Ring, синхронизировать данные по токену и общаться с умным тренером.

### Вариант Б: Production-сборка (оптимизированная)
Компилирует весь код в оптимизированные файлы для максимальной скорости работы.
```bash
# Сборка проекта
npm run build

# Запуск готового сервера
npm start
```
* Откройте браузер по адресу: **`http://localhost:3000`**

---

## 🌐 Вариант В: Полностью статический запуск (Идеально для GitHub Pages)

Если вы выгрузили чистый репозиторий на **GitHub Pages** (без бэкенд сервера Node.js):
1. Ваше приложение будет отлично работать прямо из браузера!
2. В интерфейсе приложения в правом верхнем углу нажмите кнопку **«Настройки API (GitHub)»**.
3. Введите ваши персональные ключи:
   * **Gemini API Key** (для аналитики и чат-бота)
   * **Oura Personal Access Token** (для мгновенной синхронизации)
4. Ваши ключи будут сохраняться исключительно в вашем веб-браузере (`localStorage`) и напрямую отправлять запросы в официальные API Google и Oura абсолютно бесплатно и безопасно!
